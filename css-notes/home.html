CSS TOPICS

Layout

Layout is the soul of CSS. Understanding layout modes is essential for a solid foundation in CSS. Learn about the various positioned layouts: absolute, relative, and more. Then delve into the coolest modes CSS has to offer: Flexbox and Grid.

Responsive Design

Modern web design involves responsive web pages that can transform smoothly from mobile to desktop screen sizes and everything in between smoothly and efficiently. Learn how to utilize the media query to do just that.

Selectors & Variables

Choose how to target your html by selecting the element, a Class, or Id. There are many ways to combine selectors to increase specificity. Use variables for your color palette to decrease repetition when changing styles.

Units, Height & Width

Learn when to use pixels versus rem, and why. Also learn a few tips and tricks when working with height and width.
