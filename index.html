Check out my projects below:

COCONUT CRAVINGS

Coconut Cravings is a full-stack, fully responsive Next.js recipe website. It features an SQLite database with dynamic page generation and nutrition info generated from an external api. Users can comment and rate recipes as well as double recipes and change measurements from US to metric. Technologies used include React, Next.js, SQLite, Prisma, Markdown, Git, and CSS modules.

ALL THINGS JAVASCRIPT

All Things JavaScript is a fully responsive site I created as I was learning JavaScript. It includes explanations of many JavaScript topics including the basics of data types, array methods, classes, and delves into more advanced concepts of asynchronous JavaScript, promises, and more.

THE COOL PARTS OF CSS

The Cool Parts of CSS is a fully responsive site that highlights core CSS concepts, including layout types such as flow, flex, and grid. It also explains media queries and techniques for responsive design.
